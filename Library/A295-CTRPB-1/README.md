# USB Type-C コネクタ (A295-CTRPB-1) - KiCad ライブラリ

## 概要

秋月電子通商で購入できるUSB Type-Cコネクタ「A295-CTRPB-1」(Chang Enn製)のKiCad用シンボルとフットプリントです。

## 製品情報

- **型番**: A295-CTRPB-1
- **メーカー**: Chang Enn Co., Ltd.
- **販売**: 秋月電子通商
- **商品コード**: 116895
- **実装方式**: スルーホール (Through-Hole)
- **用途**: USB Type-C 電源供給専用

## 仕様

| 項目 | 仕様 |
|------|------|
| コネクタタイプ | USB Type-C レセプタクル (メス) |
| ピン数 | 2ピン (VBUS, GND) + シールド |
| 定格電流 | VBUS: 3A, GND: 1.25A |
| 定格電圧 | 5V DC |
| 実装方式 | スルーホール (Through-Hole) |
| 基板からの高さ | 約3.06mm |
| コネクタ幅 | 8.85mm |
| コネクタ奥行き | 6.50mm |
| 動作温度範囲 | -20°C ~ +85°C |

## ピン配置

このコネクタは**電源専用**で、データ通信用のピンはありません。

| ピン番号 | 信号名 | 説明 |
|---------|--------|------|
| 1 | VBUS | 電源 (+5V, 最大3A) |
| 2 | GND | グランド (最大1.25A) |
| SH | SHIELD | シールド/機械的固定 (4箇所) |

## フットプリントの特徴

### パッド配置

データシートの「RECOMMEND P.C.B LAYOUT」を基に、JLCPCBょ2層基板のデザインルール（アニュラーリング幅 ≥ 0.18mm）に対応した設計:

- **Pin 1 (VBUS/Positive)**: 
  - 位置: X=-1.815mm, Y=3.79mm
  - パッドサイズ: 0.96mm × 0.71mm (横長角丸長方形)
  - ドリル: 0.6mm × 0.35mm (楕円、横長)
  - アニュラーリング幅: 0.18mm
  - タイプ: スルーホール

- **Pin 2 (GND/Negative)**: 
  - 位置: X=+1.815mm, Y=3.79mm
  - パッドサイズ: 0.96mm × 0.71mm (横長角丸長方形)
  - ドリル: 0.6mm × 0.35mm (楕円、横長)
  - アニュラーリング幅: 0.18mm
  - タイプ: スルーホール

- **シールド (SH)**: 
  - 2箇所の縦長角丸長方形スルーホール
  - 位置: (±4.3mm, 0mm)
  - パッドサイズ: 0.71mm × 1.24mm (縦長角丸長方形)
  - ドリル: 0.35mm × 0.88mm (楕円、縦長)
  - アニュラーリング幅: 0.18mm

### 寸法の詳細

- **VBUS-GND間隔**: 3.63mm (中心間距離)
- **Shield間隔**: 8.6mm (中心間距離)
- **垂直オフセット**: 3.79mm (ShieldとVBUS/GND間)

> [!NOTE]
> **JLCPCB対応**
> - パッドサイズはJLCPCBの2層基板デザインルールに準拠
> - アニュラーリング幅: 最小0.18mm（JLCPCB要求値）
> - データシートの推奨値より若干大きめですが、製造性を確保しています

### 機械的固定

2箇所のシールドスルーホールにより、コネクタを基板にしっかりと固定します。これにより、USB抜き差し時の機械的ストレスに耐えられます。

## KiCadでの使用方法

### 1. ライブラリの追加

#### シンボルライブラリ

```
1. KiCadプロジェクトを開く
2. Preferences → Manage Symbol Libraries
3. Project Specific Libraries タブを選択
4. フォルダアイコンをクリック
5. "esp32/Aki-USB-C-Connector/A295-CTRPB-1.kicad_sym" を選択
6. Nickname: "Aki-USB-C-Connector" と入力
7. OK をクリック
```

#### フットプリントライブラリ

```
1. Preferences → Manage Footprint Libraries
2. Project Specific Libraries タブを選択
3. フォルダアイコンをクリック
4. "esp32/Aki-USB-C-Connector/A295-CTRPB-1.pretty" フォルダを選択
5. Nickname: "Aki-USB-C-Connector" と入力
6. OK をクリック
```

### 2. 回路図での使用

1. シンボルピッカーを開く (ショートカット: `A`)
2. "A295-CTRPB-1" を検索
3. 配置してピンを接続:
   - **Pin 1 (VBUS)**: +5V電源ラインへ
   - **Pin 2 (GND)**: グランドへ
   - **Pin SH (SHIELD)**: グランドへ (推奨)

### 3. 基板設計時の注意事項

#### 配線推奨事項

- **VBUS配線**: 
  - 最低1.5mm幅 (3A対応の場合)
  - 2mm以上推奨 (余裕を持たせる)
  - 銅箔厚: 35μm (1oz) 以上

- **GND配線**: 
  - ベタGNDを使用
  - シールドピンもGNDに接続

#### クリアランス

- コネクタ周辺5mm以内に高さのある部品を配置しないでください
- USBケーブル挿入スペース(コネクタ前方10mm以上)を確保してください
- 基板端からの距離: 最低2mm以上

#### 基板固定

- シールドピンのスルーホールは基板の機械的強度を高めます
- 基板厚: 1.6mm推奨

## 回路例

### 基本的な5V電源入力回路

```
USB Type-C Connector (J1: A295-CTRPB-1)
├─ VBUS (Pin 1) ──→ +5V
│                    │
│                    ├─→ C1 (10μF) → GND  (バイパスコンデンサ)
│                    ├─→ F1 (PTC)  → 回路へ (過電流保護、オプション)
│                    └─→ D1 (逆接保護ダイオード、オプション)
│
├─ GND (Pin 2) ──→ GND
│
└─ SHIELD (SH) ──→ GND
```

### 必要な外付け部品

| 部品 | 値 | 用途 | 必須/推奨 |
|------|-----|------|----------|
| C1 | 10μF~100μF | 電源デカップリング | 必須 |
| F1 | PTC (3A) | 過電流保護 | 推奨 |
| D1 | ショットキーダイオード | 逆接保護 | オプション |

> [!IMPORTANT]
> **電源専用コネクタ**
> - このコネクタはデータ通信に対応していません
> - USB Type-C規格のCC (Configuration Channel) ピンがないため、一部のUSB電源では動作しない可能性があります
> - 標準的なUSB ACアダプタやモバイルバッテリーでは問題なく使用できます

> [!NOTE]
> **USB Type-C規格について**
> - CC1/CC2ピンがないため、USB Power Delivery (USB-PD) には非対応です
> - 5V固定電源としてのみ使用できます
> - 最大電流は3Aですが、実際の供給電流は電源側の仕様に依存します

## 実装時の注意事項

#### はんだ付け

#### 手はんだ推奨

スルーホール実装のため、手はんだが推奨されます。

1. **温度**: 300-350°C
2. **手順**:
   - Pin 1 (VBUS) とPin 2 (GND) を先にはんだ付け
   - 表面と裏面の両方からはんだを流し込む
   - シールドピン(SH)を最後にはんだ付け
   - 各ピンを確実にはんだ付けして機械的強度を確保

#### リフロー（非推奨）

このコネクタはスルーホールタイプのため、リフロー実装には適していません。

### 検査項目

- [ ] VBUS/GNDのスルーホールに表裏両面からはんだが十分に付いているか
- [ ] ブリッジ(ショート)がないか
- [ ] 2箇所すべてのシールドピンがしっかり固定されているか
- [ ] コネクタが傾いていないか
- [ ] コネクタが基板から浮いていないか

## トラブルシューティング

### Q1: USB電源を接続しても電圧が出ない

**A**: 
- はんだ付けを確認 (特にVBUS/GNDパッド)
- USB電源側の仕様を確認 (一部のUSB-PD専用電源では動作しない可能性)
- テスターでVBUS-GND間の電圧を測定 (5V前後であるべき)

### Q2: コネクタが基板から浮いている

**A**:
- シールドピンのはんだ付けを確認
- リフロー温度が適切か確認
- パッドサイズが正しいか確認
- 手はんだの場合、十分な予熱を行う

### Q3: USBケーブルの抜き差しでコネクタが外れる

**A**:
- 2箇所すべてのシールドピンがしっかりはんだ付けされているか確認
- 基板厚が薄すぎないか確認 (1.6mm推奨)
- 基板の固定方法を見直す

### Q4: 一部のUSB電源で動作しない

**A**:
- このコネクタはCC (Configuration Channel) ピンがありません
- USB-PD専用の電源では動作しない可能性があります
- 標準的な5V USB ACアダプタやモバイルバッテリーを使用してください
