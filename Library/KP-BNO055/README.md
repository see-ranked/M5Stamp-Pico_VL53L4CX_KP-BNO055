# KP-BNO055 KiCad Library

このディレクトリには、ワンダーキット製 KP-BNO055 9軸センサーモジュール用のKiCadライブラリファイルが含まれています。

## ファイル構成

- `KP-BNO055.kicad_sym` - 回路図シンボルファイル
- `KP-BNO055.pretty/KP-BNO055.kicad_mod` - フットプリントファイル

## モジュール仕様

### 基本仕様
- **搭載センサー**: Bosch Sensortec BNO055
- **測定軸**: 9軸（加速度3軸 + ジャイロ3軸 + 磁気コンパス3軸）
- **電源電圧**: DC 3.3V～5V
- **消費電流**: 動作時平均 約12mA
- **インターフェース**: I2CまたはUART（選択可能）
- **信号電圧**: DC 3.3V～5V（電源電圧と同じ）
- **基板サイズ**: 25(W) × 20(D) × 2.6(H) mm

### ピン配置

#### CN1（メインコネクタ - 4ピン）
| ピン番号 | ピン名 | 説明 |
|---------|--------|------|
| 1 | VBUS | 電源＋（DC 3.3V～5V） |
| 2 | SCL/RX | I2Cクロック または UART受信データ |
| 3 | SDA/TX | I2Cデータ または UART送信データ |
| 4 | GND | 電源− |

#### CN2（補助コネクタ - 2ピン、オプション）
| ピン番号 | ピン名 | 説明 |
|---------|--------|------|
| 5 | RESET | リセット入力 |
| 6 | INT | 割り込み出力 |

### インターフェース設定

- **I2C/UART切り替え**: 基板上のジャンパーJ3で選択
- **I2Cアドレス**: 
  - デフォルト: 0x29
  - J1ジャンパー短絡時: 0x28

## KiCadでの使用方法

### 1. シンボルライブラリの追加

1. KiCadプロジェクトを開く
2. メニューから「設定」→「シンボルライブラリを管理」を選択
3. 「プロジェクト固有のライブラリ」タブを選択
4. 「フォルダアイコン」をクリックして、`KP-BNO055.kicad_sym`ファイルを追加
5. ニックネームを「KP-BNO055」に設定

### 2. フットプリントライブラリの追加

1. メニューから「設定」→「フットプリントライブラリを管理」を選択
2. 「プロジェクト固有のライブラリ」タブを選択
3. 「フォルダアイコン」をクリックして、`KP-BNO055.pretty`フォルダを追加
4. ニックネームを「KP-BNO055」に設定

### 3. 回路図での使用

1. 回路図エディタで「シンボルを追加」ツールを選択
2. 「KP-BNO055」ライブラリから「KP-BNO055」シンボルを選択
3. 配置して配線

### 4. フットプリントの割り当て

- シンボルには自動的にフットプリント「KP-BNO055:KP-BNO055」が割り当てられています
- 必要に応じて「フットプリントを割り当て」ツールで確認・変更可能

## 接続例（I2Cモード）

```
KP-BNO055          マイコン
---------          -------
VBUS    ------>    3.3V or 5V
SCL/RX  ------>    I2C SCL
SDA/TX  ------>    I2C SDA
GND     ------>    GND
INT     ------>    GPIO（オプション）
RESET   ------>    GPIO（オプション）
```

## 測定範囲

- **加速度**: 各軸最大 ±16g
- **ジャイロスコープ**: 各軸最大 2000°/秒
- **磁気コンパス**: 
  - X,Y軸: 最大 ±1300μT
  - Z軸: 最大 ±2500μT
